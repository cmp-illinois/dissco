# DISSCO-2.2.0
Major update to DISSCO focusing on cross-platform compatibility and improved usability.

Please refer to the `BUILDING.md` file relevant to the operating system you wish to install on.

# Contributing

Thank you for your interest in contributing to the dissco project. 

You can find existing issues in the [Issues](https://github.com/cmp-illinois/dissco/issues?q=sort%3Aupdated-desc+is%3Aissue+is%3Aopen+) tab for this repo. Pull requests can be submitted for any of these issues (for particularly involved issues, please [make an issue report](https://github.com/cmp-illinois/dissco/issues/new) before attempting a solution and submitting a PR; for fairly straightforward fixes (e.g., fixing typos or compiler warnings), an issue report is not necessary).

Please [fork this repo](https://github.com/cmp-illinois/dissco/fork) and make changes to your fork to submit a PR. Please name the branch you make your changes on something informative that succinctly describes the subject of the request.
